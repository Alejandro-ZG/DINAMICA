# Publicar en GitHub Pages

Repositorio: `Alejandro-ZG/DINAMICA`

## Opción con Git

Desde una carpeta donde tengas estos archivos:

```bash
git clone https://github.com/Alejandro-ZG/DINAMICA.git
cd DINAMICA
# copia aquí los archivos de esta app, dejando index.html en la raíz
git add .
git commit -m "Agregar Espejo Bíblico"
git push origin main
```

## Activar GitHub Pages

1. Entra al repositorio en GitHub.
2. Ve a `Settings > Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. En `Branch`, selecciona `main` y `/root`.
5. Guarda los cambios.

El enlace esperado será:

```txt
https://alejandro-zg.github.io/DINAMICA/
```
